-------------------------------------------------
NEWTON PROJECT 
-------------------------------------------------

## Build Setup

# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and launch server
$ yarn build
$ yarn start

# generate static project
$ yarn generate

-------------------------------------------------
PAGES
-------------------------------------------------

-- INDEX --

Index.vue inside the pages folder is the main pages
is the main page of the Newton website. 

-- Institutions.vue --

Institutions.vue page is the page that shows the Institutions
that take part in the Newton project and the people that take part
in the project as well.

-- intro.vue -- 

This is the ABOUT page, it shows the Objectives, Approach and the Gannt 
Chart of the project. The Gannt Chart is derived from a component called GanntChart.vue

-- outreach.vue -- 

Useful Links from the community

-- technicaldocs.vue --

The page where someone requests the technical documents of the Newton Project.

-- datapage.vue -- 

The page where someone requests scientific data that are part of the project.

-------------------------------------------------
COMPONENTS
-------------------------------------------------

-- About.vue --

In this component we have the Objectives and the Approach fields of the About page.

-- GanntChart.vue -- 

In this component we draw our GanntChart.

-- Footer.vue -- 

We define our Footer which is displayed in all pages.

-- Header.vue --

We define our header which is displayed in all pages.
